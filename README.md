#  Sistema de Controle de Estoque em Python

Projeto desenvolvido com o objetivo de praticar lógica de programação e estruturação de sistemas utilizando Python.

##  Funcionalidades

* Cadastro de produtos
* Edição de produtos
* Exclusão de produtos
* Listagem de produtos
* Validação de dados
* Persistência em JSON

##  Tecnologias utilizadas

* Python
* JSON

##  Estrutura do projeto

* Organização modular do código
* Separação por responsabilidades (models, services, utils)

##  Como executar

```bash
python main.py
```

##  Objetivo

Evoluir habilidades em desenvolvimento back-end com Python e boas práticas de programação.
